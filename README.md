## Installation

Tiến hành build 

```sh
cd rasa-docker
```

**Build**

```sh
docker-compose build
```

**Chạy môi trường dev**
```sh
docker-compose --env-file .env.dev up -d
```

**Chạy môi trường prodution**
```sh
docker-compose --env-file .env.prod up -d
```
